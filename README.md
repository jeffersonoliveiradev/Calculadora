# Calculadora em C#

## Sobre o projeto

Este projeto é uma calculadora simples desenvolvida em C# com o objetivo de praticar os primeiros conceitos da linguagem.

Durante o desenvolvimento foram trabalhados conceitos básicos como:
- Entrada e saída de dados;
- Variáveis;
- Operadores matemáticos;
- Estruturas condicionais;
- Métodos;
- Organização básica de um programa em C#.

## Funcionalidades

A calculadora permite realizar operações básicas:

- Soma;
- Subtração;
- Multiplicação;
- Divisão.

## Tecnologias utilizadas

- C#
- .NET Console Application

## Objetivo

O objetivo deste projeto foi colocar em prática meus primeiros estudos em C#, criando uma aplicação simples para entender a lógica de programação e a estrutura de uma aplicação em console.

Este é um projeto de aprendizado e faz parte da minha evolução no desenvolvimento de software.
